## Task 1: Add content card wrapper in ScreenPlayer

**File:** `src/components/ScreenPlayer.tsx`

The current structure inside the `<AnimatePresence>` block (lines 134-148):
```tsx
{showUI && (
  <motion.div {...uiReveal} className="flex w-full flex-col items-center">
    <QuestionPrompt screenId={screen.id} visible={true} />
    <UIInput ... />
  </motion.div>
)}
```

Change to wrap QuestionPrompt + UIInput in a conditional content card:

```tsx
{showUI && (
  <motion.div {...uiReveal} className="flex w-full flex-col items-center">
    {screen.ui !== "none" && screen.ui !== "start-button" && screen.ui !== "submit-button" ? (
      <div
        role="region"
        aria-label="Survey question"
        className="w-full max-w-md rounded-2xl border border-[--card-border] bg-[--card] px-5 py-6 backdrop-blur-sm overflow-y-auto max-h-[70svh]"
      >
        <QuestionPrompt screenId={screen.id} visible={true} />
        <UIInput
          key={`${screen.id}:${JSON.stringify(initialValue ?? null)}`}
          type={screen.ui}
          config={screen.uiConfig}
          initialValue={initialValue}
          onSubmit={onComplete}
        />
      </div>
    ) : (
      <>
        <QuestionPrompt screenId={screen.id} visible={true} />
        <UIInput
          key={`${screen.id}:${JSON.stringify(initialValue ?? null)}`}
          type={screen.ui}
          config={screen.uiConfig}
          initialValue={initialValue}
          onSubmit={onComplete}
        />
      </>
    )}
  </motion.div>
)}
```

**Key constraints:**
- The `motion.div` with `uiReveal` animation stays as the outer wrapper (animation unchanged)
- Content card is a static `div` inside the animated container
- `overflow-y-auto max-h-[70svh]` handles landscape orientation (P2-1)
- `role="region" aria-label="Survey question"` for accessibility

## Task 2: Redesign back button

**File:** `src/components/ScreenPlayer.tsx`

Change the back button (lines 124-131) from:
```tsx
<button
  onClick={onBack}
  className="absolute left-4 bottom-8 safe-bottom z-20 min-h-[48px] rounded-full bg-white/10 px-4 py-2 text-sm text-white/70 backdrop-blur-sm hover:bg-white/20 flex items-center"

```

To:
```tsx
<button
  onClick={onBack}
  className="absolute left-4 bottom-8 safe-bottom z-20 min-h-[48px] rounded-full bg-black/70 px-4 py-2 text-sm text-white border border-white/10 backdrop-blur-sm hover:bg-black/80 flex items-center gap-2"
```

Changes:
- `bg-white/10` → `bg-black/70` (dark glass, matching nav controls)
- `text-white/70` → `text-white` (full opacity)
- Add `border border-white/10` (subtle edge definition)
- `hover:bg-white/20` → `hover:bg-black/80`
- Add `gap-2` (future-proofs for icon+text layout per UI-SPEC contract)
