# Good Morning, Nikhil - Visual Style Guide

## Art Direction Philosophy

**Visual Style**: Painterly-Cinematic

This is a cinematic broadcast television experience, NOT an arcade game. Every visual element should evoke pre-recorded, high-fidelity television footage with the warmth and depth of professional studio production.

### Core Principles

1. **Broadcast Authenticity**: Emulate the look of actual daytime TV shows, not mobile game interpretations
2. **Cinematic Depth**: Use atmospheric perspective, subtle depth of field, and layered lighting
3. **Painterly Texture**: Soft brushwork feel, not hard vector edges or cartoon outlines
4. **Warm Broadcast Tone**: CRT-era warmth with modern clarity - slight bloom, gentle vignettes
5. **Overlay Compatibility**: Backgrounds must support dust motes, film grain, and light flicker overlays

## Cinematic Lighting Approach

### Three-Point Lighting Standard

All scenes should feel like they use professional three-point lighting:

1. **Key Light** (Primary): 
   - 45-degree angle from subject
   - Creates main illumination and defines form
   - Temperature varies by show theme (see palettes above)

2. **Fill Light** (Secondary):
   - Opposite side from key, softer intensity (40-60% of key)
   - Reduces harsh shadows without eliminating depth
   - Often warmer than key light

3. **Back/Rim Light** (Separation):
   - Behind subject, creates edge definition
   - Separates subject from background
   - Can be colored for dramatic effect

### Lighting Rules by Context

| Context | Key Temp | Fill Ratio | Rim Style |
|---------|----------|------------|-----------|
| Game Show Stage | 4500K | 50% | Warm amber halo |
| Romantic Setting | 3000K | 70% | Soft golden edge |
| Industrial/Business | 5600K | 40% | Cool steel rim |
| Outdoor/Tribal | 2700K | 30% | Moonlight blue rim |
| Talk Show Studio | 5000K | 60% | Neutral separation |
| Commercial/Product | 5500K | 80% | Clean white edge |

### Atmospheric Effects

- **Depth haze**: Subtle atmospheric perspective, backgrounds 10-15% desaturated
- **Lens bloom**: Gentle highlight bloom around bright sources (not glow effects)
- **Vignette**: Natural lens falloff, darkening edges 15-25%
- **Film grain**: Subtle texture overlay (opacity 3-8%)
- **Dust motes**: Floating particles in light shafts where appropriate

## Character Style Guidelines

### The TV Talent Aesthetic

Characters and hosts should feel like actual television personalities, not animated mascots.

**Key Characteristics**:
- **Professional grooming**: Clean, polished appearance appropriate to show context
- **Natural proportions**: Human anatomy, no exaggerated features
- **Broadcast makeup**: Visible but not theatrical - matte skin, defined features
- **Wardrobe authenticity**: Show-appropriate attire (suits for Shark Tank, casual for Survivor)
- **Expression range**: Genuine emotional range, not cartoon reactions

**Rendering Style**:
- Soft edge work, avoiding hard outlines
- Subtle skin texture, not plastic smooth
- Natural color gradients in skin tones
- Catchlights in eyes for life
- Environmental lighting reflected on skin/clothing

**What to AVOID**:
- Cartoon proportions (big heads, tiny bodies)
- Flat cel-shading
- Neon or oversaturated colors
- Hard black outlines
- Emoji-style expressions
- Mobile game character tropes

## Implementation Notes

### Overlay Compatibility

All backgrounds must work with these animated overlays:
- Dust motes (white, 20-40% opacity, slow drift)
- Film grain (monochrome, 3-8% opacity)
- Light flicker (subtle luminosity variation, 2-5%)
- Scan lines (optional, very subtle)
- Lens flare (source-motivated only)

### Text Legibility Requirements

- Backgrounds should be dark enough (< 40% luminosity) for white text
- Or provide designated "safe zones" for text placement
- Avoid high-contrast patterns that compete with text
- Include subtle vignettes to naturally darken text areas

### Asset Specifications

- **Backgrounds**: 1920x1080 minimum, 16:9 aspect ratio
- **Format**: WebP at 85% quality for web delivery
- **Color space**: sRGB for web consistency
- **Transparency**: PNG where overlay compositing needed
